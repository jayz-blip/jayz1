# 빠른 테스트 가이드

## 백엔드가 실행 중인지 확인

### 1단계: 백엔드 상태 확인

브라우저에서 다음 URL 접속:
```
http://localhost:8000
```

**정상 응답:**
```json
{"message":"사내용 채팅 AI API","status":"running"}
```

이 응답이 보이면 백엔드가 정상 실행 중입니다! ✅

---

## 프론트엔드 실행 및 테스트

### 2단계: 프론트엔드 실행 (새 터미널/PowerShell 창)

**새 PowerShell 창을 열고:**

```powershell
cd C:\Users\malgn\Desktop\malgpt
npm run dev
```

**정상 실행 시:**
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
```

### 3단계: 브라우저에서 테스트

1. 브라우저에서 `http://localhost:3000` 접속
2. 채팅창이 보이면 성공! 🎉

## 문제 해결

### 백엔드가 응답하지 않을 때

1. **백엔드 PowerShell 창 확인**
   - "Uvicorn running on http://0.0.0.0:8000" 메시지가 있는지 확인
   - 오류 메시지가 있는지 확인

2. **포트 확인**
   ```powershell
   netstat -ano | findstr :8000
   ```
   결과가 나오면 백엔드가 실행 중입니다.

### 프론트엔드에서 연결 오류

1. **브라우저 콘솔 확인 (F12)**
   - 오류 메시지 확인
   - Network 탭에서 `/api/chat` 요청 확인

2. **백엔드가 실행 중인지 확인**
   - `http://localhost:8000` 접속 테스트
