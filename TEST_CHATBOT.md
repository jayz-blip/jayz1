# 챗봇 테스트 가이드

## 🎉 배포 완료!

이제 챗봇을 테스트할 수 있습니다!

## 🧪 테스트 방법

### 1단계: Cloudflare Pages URL 접속

1. **브라우저에서 접속**
   - `https://jayz1.pages.dev`

2. **페이지 로드 확인**
   - 채팅 인터페이스가 정상적으로 표시되는지 확인

### 2단계: 채팅 테스트

1. **간단한 질문부터 시작**
   - 예: "안녕하세요"
   - 예: "PPM에 대해 알려주세요"
   - 예: "고객 문의 처리 방법"

2. **응답 확인**
   - AI가 답변을 생성하는지 확인
   - 응답 시간 확인
   - 응답 내용이 관련 있는지 확인

### 3단계: 브라우저 개발자 도구 확인

1. **F12 키를 눌러 개발자 도구 열기**

2. **Console 탭 확인**
   - 오류 메시지가 없는지 확인
   - API URL이 올바른지 확인: `🔗 API URL: /api`

3. **Network 탭 확인**
   - `/api/chat` 요청 확인
   - 상태 코드 확인:
     - ✅ 200: 성공
     - ❌ 500: 서버 오류
     - ❌ 기타 오류

## 🔍 예상 동작

### 정상 작동 시

1. **질문 입력**
   - 채팅창에 질문 입력
   - "🚀" 버튼 클릭 또는 Enter 키

2. **로딩 표시**
   - "⏳" 버튼으로 변경
   - 타이핑 인디케이터 표시

3. **응답 수신**
   - AI 응답 표시
   - 참고 자료 개수 표시 (있는 경우)

### 문제 발생 시

1. **"Network Error" 메시지**
   - 브라우저 콘솔 확인
   - Network 탭에서 오류 확인

2. **"500 Internal Server Error"**
   - Network 탭 → Response 확인
   - 오류 메시지 확인

3. **응답이 없음**
   - Workers URL 확인
   - D1 데이터베이스 확인

## 🆘 문제가 있다면

### 문제 1: "Network Error"

**확인 사항**:
- 브라우저 콘솔 오류 확인
- Network 탭에서 요청 상태 확인
- `BACKEND_URL` 환경 변수 확인

### 문제 2: "500 Internal Server Error"

**확인 사항**:
- Network 탭 → Response 탭에서 오류 메시지 확인
- Workers URL 직접 테스트
- Workers 로그 확인

### 문제 3: 응답이 느림

**정상적인 현상**:
- 첫 요청은 시간이 걸릴 수 있음
- 임베딩 생성 및 유사도 계산 시간 소요
